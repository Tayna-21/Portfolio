  • Online Store

    This is an online store, wich includes dynamic routing, form handling and state management. The application leverages React Router for navigation across shopping cart and checkout flow. Form handling is managed with Formik and Yup. The app uses Redux Toolkit for global state management, ensuring that cart data, product details and user preferences are synchronized across the app. Cart contents is stored in Local Storage, allowing for a smooth user experience even after page refreshes

    GitHub: https://github.com/Tayna-21/Online-Store

  • UI kit

    This UI Kit is a collection of reusable components. Built with Styled Components and CSS Modules. The kit is fully documented using Storybook, making it easy to explore, test, and implement components in various projects. Date-picker is integrated with "date-fns" library, Select was built with "react-select" and Color-picker is powered by "react-colour-sketch”. All components are tested using Jest and React Testing Library.

    GitHub: https://github.com/Tayna-21/UI-kit

  • Todo App

    This project is a fully functional To-Do List application. In this task I shouldn’t use any of state management library, only client-server communication. The application includes the following features:
    - Add, delete and update To-Do items
    - All to-do items are saved in a MongoDB database
    - The backend server is created using the Express library. It provides API endpoints to handle CRUD operations for the To-Do items
    - Data fetching with TanStack React Query
    - Unit tests and integration tests are written using Jest and React Testing Library

    GitHub: https://github.com/Tayna-21/To-Do-List
